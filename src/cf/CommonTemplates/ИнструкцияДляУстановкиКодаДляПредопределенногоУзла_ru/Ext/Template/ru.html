Для исправления ошибки перейдите в периферийную информационную базу. 
Откройте форму настройками синхронизации. 
Нажав на команду Меню выберите Новый код предопределенного узла...
Если в конфигурации есть несколько планов обменов поддерживающих
универсальный формат обмена "Enterprise Data" то будет открыт список
для выбора плана обмена, у которого необходимо поменять код. После
выбора плана обмена откроется форма для изменения кода узла.

Ознакомьтесь с информацией на форме и нажмите Установить новый код.

              
После этого повторите синхронизацию (или создайте новую настройку обмена).
