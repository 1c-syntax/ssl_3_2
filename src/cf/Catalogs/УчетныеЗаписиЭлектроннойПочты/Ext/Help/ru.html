Предназначен для ведения электронной почты, которая используется для получения и отправки электронных сообщений.

Список можно открыть соответствующей командой в разделе Администрирование - Органайзер.

В списке выводится:

  • Наименование почты;
  • Имя пользователя - указывается имя отправителя писем, оно может отличаться от имени пользователя в приложении;
  • Адрес электронной почты в формате name@server.net.

Для электронной почты, получение и отправка с которой выполняется автоматически без участия пользователя, должен быть обязательно введен пароль. Также для корректной работы приложения обязательно должна быть настроена электронная почта.

Настройка почты для рассылок уведомлений

  • Приложение поставляется с предопределенной настройкой электронной почты, которая предназначена для рассылки различных уведомлений из приложения. Например, для автоматической отправки сообщений администратору по завершению обновления приложения. Для этого предварительное необходимо настроить ее.
  • Настройка основной почты для рассылок уведомлений производится по соответствующей команде в разделе Администрирование - Органайзер. Также можно Изменить ее в списке.

Настройка электронной почты

  • При попытке сохранения приложение проверяет введенную электронную почту (пытается отослать пробное сообщение). Электронная почта, не прошедшая проверку, не сохраняется. 

  • Нажмите Создать, откроется пошаговый помощник:
    • Укажите Адрес почты в формате name@server.net и нажмите кнопку Далее;
    • Выполните настройки, которые используются при работе с учетной записью:
      • Укажите Имя отправителя (Ваше имя или название организации);
      • Вариант использования - с помощью переключателя укажите, кем будет использоваться почта:
        • Всеми пользователями (общая почта)
        • Только мной.
      • Установите флажок Использовать для получения писем, для того чтобы получать письма.
    • Установите пароль, учетная запись может потребовать пароль приложения, который отличатся от основного пароля.
  • Нажмите Далее. Будет предпринята попытка автоматической настройки и проверки почты.
  • В случае, если не удалось авторизоваться в учетной записи, то помощник откроет шаг с рекомендациями. 
  • Также доступна детальная настройка учетной записи для этого нажмите Настроить вручную.

Настройка электронной почты

  • Нажмите Изменить, с помощью приложения произведите необходимые настройки.

Проверка настроек электронной почты

  • Выделите электронную почту в списке, нажмите Еще - Проверить почту для проверки введенных параметров. При этом отправляется стандартное почтовое отправление и проверяется возможность соединения с сервером входящих почтовых сообщений (если почта используется для получения сообщений).
  • Если в почте не указан пароль, он будет запрошен.
