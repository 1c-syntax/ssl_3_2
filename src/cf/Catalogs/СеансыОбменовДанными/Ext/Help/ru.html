Предназначен для фиксации информации о выполненной операции обмена (отправке или получении данных) через универсальный EnterpriseData.

В справочнике фиксируется время начала, время окончании, продолжительность операции.

Нажав кнопку Замеры производительности можно получить доступ к архиву, который содержит замеры производительности синхронизации по ключевым операциям.

Замеры производятся для следующих процедур и обработчиков:

  • ОтложенноеПроведениеДокументов
  • ДанныеXDTOИзДанныхИБ
  • ОбъектXDTOИзДанныхXDTO
  • ОбъектXDTOВСтруктуру
  • ЗаписатьОбъектВИБ
  • ОтменитьПроведениеОбъектаВИБ
  • РегистрацияОбъектов, ПакетнаяРегистрацияОбъектов (Перед выгрузкой данных)
  • Обработчики КД3: ПередКонвертацией, ПослеКонвертации, ПриОбработке и т.д.
  • ОтложенноеЗаполнениеОбъектов
  • ПоискСсылок (в регистре сведений ПубличныеИдентификаторыСинхронизируемыхОбъектов)
  • ОчисткаДвиженийДокумента

В архиве находятся два файла:

  • log.txt – список всех замеряемых операций в хронологическом порядке;
  • total.txt – итоговые данные по операциям, отсортированные по убыванию времени выполнения;
