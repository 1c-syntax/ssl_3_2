Предназначена для настройки прав доступа к сертификату.

  • Только я – при выборе сертификат будет доступен только текущему пользователю;
  • Указанные пользователи – при выборе появляется возможность настроить использование сертификата для нескольких пользователей в таблице.
    • Без использования ключа позволяет разрешить применение сертификата для операций, не требующих ключа (например, шифрования), но запрещает его использование в операциях, где ключ необходим (например, при подписании или расшифровке).
