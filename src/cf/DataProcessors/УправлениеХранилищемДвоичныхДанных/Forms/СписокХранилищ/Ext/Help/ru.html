Список хранилищ двоичных данных

Диалог предназначен для настройки хранилищ двоичных данных следующими кнопками:

  • Добавить - Позволяет добавить новое хранилище двоичных данных.
  • Настроить - Открывает диалог настройки хранилища, на котором сейчас находится указатель в таблице.
  • Удалить - Удаляет хранилище двоичных данных, на котором сейчас находится указатель в таблице.
  • Настроить использование - Открывает диалог настройки использования хранилища в зависимости от типа метаданных.

Список хранилищ двоичных данных

Данная таблица предназначена для просмотра следующей информации о хранилищах двоичных данных:

  • Имя хранилища - Имя хранилища двоичных данных.
  • Адрес хранилища - URL-адрес хранилища.
  • Режим использования - В каком режиме работает хранилище.
  • По умолчанию - Маркер хранилища по умолчанию. Гарантируется, что только одно хранилище может быть использовано как хранилище по умолчанию.
  • Минимальный размер данных для записи (байт) - минимальный размер данных в байтах, при котором он будет помещен в это хранилище.
  • Дата последней очистки - Наиболее поздняя дата и время, на которое была успешно выполнена очистка неиспользуемого места хранилища.
