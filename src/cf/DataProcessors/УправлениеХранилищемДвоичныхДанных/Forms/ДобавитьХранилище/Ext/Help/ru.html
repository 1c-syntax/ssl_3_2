Добавление внешнего хранилища двоичных данных

Диалог предназначен для добавления нового внешнего хранилища двоичных данных следующих типов:
  • Встроенное хранилище - Добавить его можно только в случае, если оно еще не добавлено в список хранилищ двоичных данных. В таком случае никаких дополнительных параметров указывать не нужно.
  • Внешнее хранилище по протоколу S3 - Внешнее S3-совместимое хранилище двоичных данных, в котором корзина указывается в пути URL.
  • Внешнее хранилище по протоколу S3 (Виртуальный хост) - Внешнее S3-совместимое хранилище двоичных данных, в котором корзина указывается в доменном имени.

Параметры настройки внешних хранилищ по протоколу S3

В каждом случае добавления внешнего хранилища по протоколу S3 необходимо указать следующие параметры подключения:

  • Имя - Уникальное название внешнего хранилища двоичных данных, не более 32 символов.
  • URL - Адрес внешнего хранилища, содержащий протокол доступа к хранилищу (http или https), а также корзину в составе пути, либо в доменном имени.
  • Идентификатор доступа - Идентификатор доступа (AccessId) к внешнему хранилищу двоичных данных.
  • Ключ доступа - Секретный ключ доступа (SecretKey) к внешнему хранилищу двоичных данных.


После заполнения всех полей появится возможность нажать кнопку "ОК" для добавления внешнего хранилища, либо "Отмена" для закрытия формы и возврата к списку менеджеров.
