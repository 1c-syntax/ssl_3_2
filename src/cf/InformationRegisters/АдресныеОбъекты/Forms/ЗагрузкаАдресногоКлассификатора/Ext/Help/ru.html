В адресном классификаторе хранятся адресные сведения, необходимые для работы в приложении.

Использование адресного классификатора помогает заполнять адреса правильно и быстро. Без этого невозможна сдача отчетности. Например, при сдаче отчетности по работникам в налоговую инспекцию, в пенсионный фонд необходимо заполнить адреса по стандарту, иначе отчетность не будет принята. Кроме того, адреса могут меняться и не всегда можно уследить за этими изменениями самостоятельно. Приложение позволяет управлять адресами и поддерживать их в актуальном состоянии.

Для всех регионов РФ имеется возможность загрузки адресных сведений с сайта поддержки пользователей приложения 1С:Предприятие фирмы "1С", а также из файлов, распространяемых ФНС РФ.

Окно для работы с адресным классификатором открывается по команде Загрузить классификатор раздела Администрирование - Интернет-поддержка и сервисы - Классификаторы и курсы валют.

Загрузка адресного классификатора

  • При загрузке классификатора можно выбрать один, несколько или все регионы России, по которым необходимо загрузить адресные сведения. Это удобно, т.к. можно не загружать адреса тех регионов, которые заведомо не будут использоваться в работе. Включите флажки рядом с наименованиями нужных регионов. Для того чтобы выбрать все регионы для загрузки, нажмите кнопку выбора всех , для того чтобы отменить выбор, нажмите кнопку отмены выбора . Для поиска регионов в списке используйте Поиск. В нижней части списка идет подсчет количества выбранных регионов.
  • Для повышения точности и скорости ввода адреса, а также для исключения опечаток рекомендуется включить загрузку сведений о помещениях-квартирах и офисах. Загрузка позволяет подбирать конкретные помещения прямо из списка в форме ввода, что гарантирует корректность формирования полного адреса и необходима для его правильной выгрузки в отчетность. При использовании веб-сервисов подбор помещений доступен сразу. При использовании загруженного классификатора для подбора помещений нужно установить флажок Загрузка помещений.
  • Загрузка адресных сведений приведет к увеличению размера адресного классификатора примерно в два раза, а для хранения данных о помещениях потребуется дополнительно до 3ГБ дискового пространства. Поэтому по умолчанию история изменения адресных объектов не загружается в приложение. При необходимости (например, в приложении вводятся паспортные данные сотрудников, в них могут быть устаревшие адреса) включите флажок Загружать историю изменений адресных объектов.
  • С помощью переключателя выберите Вариант загрузки:
    • С портала 1С:ИТС;
      • Для загрузки необходимо заполнить данные авторизации для подключения к Интернет-поддержке пользователей.
    • Из папки на компьютере.
      • Укажите путь к папке (также можно нажать клавишу F4). Скачать адресные сведения для загрузки можно на сайте 1С:ИТС - по ссылке;
  • После указания всех параметров нажмите Загрузить адресные сведения.
  • Во время загрузки приложением выводится сообщение о состоянии загрузки:
    • количество загружаемых регионов;
    • источник загрузки;
    • загружаемый в текущий момент регион.
  • После успешной загрузки выводится сообщение. Нажмите Закрыть для возврата в приложение.
  • При повторной загрузки адресных сведений, ранее загруженных в классификатор, они перезаписываются.
